# Bookle

Bookle adalah sebuah website manajemen servis menggunakan AI chatbot.

## Tech Stack

- Nextjs
- Drizzle ORM
- PostgresSQL
- Gemini AI
- Shadcn UI

## Features

### Admin Dashboard
Merupakan sebuah layout dengan sidebar dan menu yang dibangun menggunakan shadcn.
Dapat login super admin yang dapat melihat semua data dan melakukan manajemen data.
Sedangkan akun admin demo dikenai rate limit

### Manajemen Servis

User bisa tambah jadwal servis, terdiri dari nama, harga, foto, dan jam.

### Manajemen Chatbot

Setup RAG yang dibutuhkan untuk chatbot menjawab pertanyaan anda.

Proyek yang dibuat akan bersifat demo dan akan reset setiap minggunya dan juga memiliki rate limit
sebanyak 10 perubahan AI chatbot per hari jika untuk akun demo
